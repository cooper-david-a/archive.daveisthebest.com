What is this all about?

Really, I'm just having fun learning about web development. I started learning html and css in the spring of 2021 and started building this site a few months later to guide my learning.

At this point, I plan to build everything using pure html, css, and django. It's not easy but I think it will pay off in better understanding. Later, I'll look into css frameworks like bootstrap.
