Sudoku Solver

This is a sudoku solver that I first implemented in python and converted to javascript to run in the browser. It uses OpenCv for the computer vision tasks, a neural network trained in scikit learn and deployed using onnx-runtime to recognize the puzzle. This solver runs completely on the client side. This is still a work in progress, all the pieces work but the UI needs some work and it probably doesn't work on all browsers or platforms.

Instructions

  1. Put a sudoku puzzle in the video frame
  2. Press Go
  3. Verify the digit recognition in the bottom image
  4. Press Stop to solve and display the solution
